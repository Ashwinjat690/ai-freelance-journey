# 🤖 Buddy - My AI Chatbot

**Live Demo:** [https://ai-freelance-journey-jipmlr5sxpc7pnbzvx6ocn.streamlit.app/](https://ai-freelance-journey-jipmlr5sxpc7pnbzvx6ocn.streamlit.app/)

### About Project
I built "Buddy" - a friendly, human-like AI chatbot during my AI Freelance Learning Journey. It can help with studies, coding doubts, motivation, and casual conversations.

### Tech Stack
- **Python**
- **Streamlit** (for web interface)
- **LangChain** + **Groq** (Llama 3.1 model)
- Deployed on Streamlit Cloud

### Features
- Personalized (remembers your name)
- Friendly & natural conversation
- Remembers chat history in session
- Easy to use interface

- Building web apps with Python
- Integrating AI models using APIs
